# Portpolio for Neon K.I.D

This repository is my portfolio repository. Used React.js library was developed based SPA



## Build CSS (Required)

```
$ yarn build-css
```

This is a command that changes the scss code to css.



## Production

```
$ yarn build
```



## Run (Local)

```
$ yarn install
$ yarn start
```
